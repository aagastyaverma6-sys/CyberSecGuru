Firewall

Category: Network Security Difficulty: Beginner Related: IDS, Zero Trust

Definition

A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined rules.

Why it matters: Firewalls act as the first line of defense against unauthorized access.

Types

  • Packet-filtering: Basic rule-based filtering.
  • Stateful inspection: Tracks active connections.
  • Next-gen firewalls: Include intrusion prevention and application awareness.

Best practices

  • Regularly update firewall rules.
  • Segment networks with internal firewalls.
  • Monitor logs for anomalies.
