Sandboxing

Category: Containment Difficulty: Intermediate Related: Ransomware, Vulnerability

Definition

Sandboxing isolates applications or code execution in a controlled environment to prevent malicious activity from affecting the wider system.

Why it matters: It allows safe testing of suspicious files or programs.

Examples

  • Running untrusted code in a virtual machine.
  • Email security gateways analyzing attachments.
  • Browser sandboxes isolating web processes.

Best practices

  • Use sandboxing for malware analysis.
  • Combine with intrusion detection systems.
  • Regularly update sandbox environments.
