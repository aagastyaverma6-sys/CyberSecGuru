Intrusion Detection System (IDS)

Category: Monitoring Difficulty: Intermediate Related: Firewall, Zero Trust

Definition

An IDS is a system that monitors network or system activities for malicious actions or policy violations.

Why it matters: IDS helps detect attacks early, reducing potential damage.

Types

  • Network-based IDS: Monitors traffic across the network.
  • Host-based IDS: Monitors activity on individual devices.

Best practices

  • Combine IDS with firewalls and SIEM tools.
  • Regularly update detection signatures.
  • Investigate alerts promptly.
