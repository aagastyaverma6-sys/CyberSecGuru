Authorization

Category: Access Control Difficulty: Beginner Related: Authentication, Least Privilege

Definition

Authorization determines what an authenticated user is allowed to do, such as accessing files, executing commands, or modifying data.

Why it matters: Even authenticated users should only have access to resources they are permitted to use.

Examples

  • Role-based access control (RBAC).
  • File permissions in operating systems.
  • API scopes in cloud services.

Best practices

  • Apply least privilege principles.
  • Regularly audit permissions.
  • Use centralized access management.
