SQL Injection

Category: Web Security Difficulty: Intermediate Related: CSRF, Vulnerability

Definition

SQL Injection is a web attack where malicious SQL statements are inserted into input fields to manipulate a database.

Why it matters: SQL Injection can expose or destroy sensitive data and compromise entire applications.

Examples

  • Bypassing login forms with `' OR '1'='1' --`.
  • Extracting entire user tables.
  • Deleting records via injected queries.

Best practices

  • Use parameterized queries (prepared statements).
  • Validate and sanitize user input.
  • Limit database privileges for applications.
